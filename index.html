University of Derby

Introduction

A static html build of the University of Derby website components, to be used as a starting point for conversion into CMS-specific templates.

Templates are authored using twig syntax, stylesheets are authored with SCSS syntax, Javascript is transpiled using Babel and Webpack. A static preview website is generated using Fractal and is hosted online at https://preview.deeson.net/derby/
