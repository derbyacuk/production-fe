    // if we haven't scrolled, exit the function early.

Pattern Hero

Pattern hero elements encapsulate a header and menu, optional breadcrumb navigation, optional call to action button and optional page tagging, a pattern, heading and lead content paragraph.

The pattern is displayed at a minimum size of 100% viewport height with the lead content paragraph overlapping beyond the height of the viewport.

Key Features

  • Full bleed pattern with customisable colours
  • Optional page type tag
  • Optional breadcrumbs navigation
  • Optional Call to Action button
